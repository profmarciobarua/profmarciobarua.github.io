CSS

Para mudar a aparência de uma página web é necessário utilizar uma linguagem de estilo chamada CSS (Cascading Style Sheets - Folha de Estilo em Cascata). Com ela podemos mudar o visual dos elementos que compõem um documento HTML.

HTML

Serve para estruturar uma página web.

CSS

Serve para estilizar uma página web.

Reponsável pela Aparência da Página WEB.

Exemplo código CSS e HTML

Significa poder alterar o tipo da variável ao longo do código.

<div> <a href="#">Contato</a> </div>
Código HTML
a { text-decoration: none; color: white; background-color: black; padding: 10px 15px; }
